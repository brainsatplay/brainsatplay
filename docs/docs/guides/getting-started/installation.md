# Installation
To get started, import `brainsatplay` into your project:

#### Browser
##### Script Tag
``` html
<script src="https://cdn.jsdelivr.net/npm/brainsatplay@latest">
```

#### ES6
``` js
import * as brainsatplay from 'brainsatplay'
```

#### Node.js
``` js
const brainsatplay = require('brainsatplay')
```
